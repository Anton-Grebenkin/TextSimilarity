# TextSimilarity
Sample WEB application project for using text similarity calculation API.
The project consists of .NET API and React UI.
## API
### Technologies
- .NET 6.0
- MS SQL Server Express
- Entity Framework Core
- MediatR
- Mapster
- Fluent Validation
- Fluent Results
## UI
### Technologies
- React
- Typescript
- Tailwind CSS
- Redux
- Redux Toolkit
